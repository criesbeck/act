A scenario canvas combines a scenario with the user stories required to support it.
A scenario canvas provides critical context for properly understanding and implement the user stories. The canvas is a more coherent, richer alternative to adding an "in order to [accomplish goal]" phrase to each story.
Prioritize stories by the scenarios they complete. Prioritize scenarios by the hypothesis testing, demoes, and viable releases they enable.
Within a scenario, do the payoff first.
